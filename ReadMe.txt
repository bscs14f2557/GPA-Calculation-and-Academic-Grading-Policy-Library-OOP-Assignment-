Title : GPA Calculation & Academic Grading Policy Library

Introduction:
This is a simple program with library calculating students grades and gpa, average results , difference of results and course information of students.
It allows for calculation of multiple students (just like real systems) and even also multiple courses of a single student.

Features:
* Include multiple students
* Add multiple courses per student
* Function Overloading ( 3 versions of addcourse function)
* Operator Overloading (+ , - ,= ,<<)
* Automatic GPA Calculation
* Grade Calculation ( of every course and overall)
* Overall Performance and Comparison

Concepts Used:
* Classes (Student , Course)
* Objects
* Constructors ( Default , Copy , Parameterized)
* Function Overloading
* Operator Overloading
* Arrays of Objects
* Encapsulation
